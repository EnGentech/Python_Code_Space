=== 5 Block Elements ===

  • <Blockquote> element

    In HTML, the <blockquote> element is used to represent a block of text that is a quotation from another source. It's a block-level element, meaning it typically starts on a new line and takes up the full width of its containing element. The purpose of <blockquote> is to provide a semantic way of indicating quoted content, such as excerpts from books, articles, or other written works. for example

    "The only way to do great work is to love what you do." - Steve Jobs

  • <Pre> element

    In HTML, the <pre> element, which stands for "preformatted text," is used to define text that should be presented exactly as it appears in the HTML code. This element is particularly useful when you want to display code snippets, ASCII art, or any content where whitespace, line breaks, and indentation are crucial and should be preserved. 

                            I want this lines of text to be displayed
                        exactly as I am formating it.     !

  • <fieldset> element

    The <fieldset> element in HTML is used to group related elements within a form and create a visual and semantic container for them. It is often used in conjunction with the <legend> element to provide a title or description for the grouped content. The primary purpose of >fieldset> is to improve the organization and accessibility of form controls. for example

  • <address> element

    In HTML, the <address> element is used to define the contact information for the author or owner of a document, or for the person or organization referenced in the content. The content inside the <address> element typically includes contact details such as physical address, email address, phone number, etc. for example

    Red-river college is located at

    160 Princess Street, Winnipeg, Manitoba, R3B 1K9

  • <nav> element

    • In HTML, the <nav> element is used to define a navigation section on a web page. It is a block-level element that is intended to contain links to other pages, sections within the same page, or other resources that help users navigate through the website. The <nav> element is particularly useful for marking up the main navigation menu of a site.

=== 3 Inline Elements ===

  1. <Span> element

    The <span> element in HTML is a generic inline container that doesn't have any inherent semantic meaning. It is primarily used to apply styles or scripting to a specific portion of text within a larger block of content. Unlike block-level elements (e.g.,

    ), which create a distinct block of content on the page, <span> is an inline element that doesn't introduce a line break and only takes up as much width as necessary.
    for example

    My name is Precious, I love Red color

  2. <u> element

    The <u> element in HTML is used to define and render text with an underline. It is traditionally associated with underlining text, making it visually distinct. However, the use of <u> for underlining text is considered somewhat outdated in modern web development. for example

  3. <a> element

    The <a> tag in HTML is used to create hyperlinks, allowing users to navigate to other web pages or resources. The "a" in <a> stands for "anchor," and it's often referred to as the "anchor element." for example
